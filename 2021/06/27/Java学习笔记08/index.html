Java学习笔记08

发表于 2021-06-27

关键字：Math类、Arrays类、System类、Date类、SimpleDateFormat类、Calendar类、LocalDate类、LocalTime类、LocalDateTime类、DateTimeFormatter类、Instant类

1.Math类

Math类提供了很多和数学计算相关的方法，且都是静态方法

  1. abs：求绝对值

  2. pow：求幂

  3. ceil：向上取整

  4. sqrt：求开方

  5. random：获取随机数[0,1)

  6. sort：排序

  7. sort：定制排序

    根据o1-o2或o2-o1决定正序和倒序，实现定制排序

    可以根据o1-o2或o2-o1决定正序和倒序，是因为在sort方法底层，会通过匿名内部类调用Comparator的compare方法，并根据compare方法的返回值决定排序的顺序

    下面是一个帮助理解sort底层调用compare方法的例子

  8. binarySearch：通过二分搜索进行查找，要求必须排好序

  9. copyOf：数组元素的复制

  10. fill：数组元素填充

  11. equals：比较两个数组元素是否完全一致

  12. asList：将一组值转换成list

3.System类

常用方法

  1. exit：退出当前程序

  2. arraycopy：复制数组元素，比较适合底层调用，一般使用Arrays.copyOf()完成复制数组

  3. currentTimeMillens：返回当前时间距离1970-1-1的毫秒数

  4. gc：运行垃圾回收机制

4.BigInteger类和BigDecimal类

BigInteger类和BigDecimal类都是用于保存大数的，BigInteger类用于保存大整数，BigDecimal用于保存高精度浮点数

常用方法

两种类的加减乘除方法用法类似

  1. add：加

  2. subtract：减

  3. multiply：乘

  4. divide：除

4.Date类

Date类是第一代日期类，JDK 1.0就包含了Date类，但是它的大多数方法已经在JDK 1.1引入Calendar类之后被弃用了

Date类通常和SimpleDateFormat类配合使用，Date获取到的时间可以再自定义显示格式，通过SimpleDateFormat的format方法格式化，已经格式化的数据，也可以通过SimpleDateFormat的parse方法再转为Date类型

5.Calendar类

Calendar类是第二代日期类，在JDK 1.1引入

Calendar类是一个抽象类，并且构造器是私有的，它通过getInstance方法获取实例

下面是Calendar类的使用

Calendar也存在很多问题

  1. 可变性：像日期和时间这样的类应该是不可变的
  2. 偏移性：Date中的年份是从1900开始的，而月份都从0开始
  3. 格式化：格式化只对Date有用，Calendar则不行
  4. 此外，它不是线程安全的；不能处理闰秒等

6.LocalDate类、LocalTime类和LocalDateTime类

LocalDate类、LocalTime类和LocalDateTime类是JDK8加入的第三代日期类

LocalDate只包含日期，可以获取日期字段

LocalTime只包含时间，可以获取时间字段

LocalDateTime包含日期和时间，可以获取日期和时间字段

使用示例

三种类获取字段的方法名一样

格式化

第三代日期类使用DateTimeFormatter类格式化，DateTimeFormatter和SimpleDateFormat用法类似

plus和minus

  1. plus：加时间

  2. minus：减时间

    ```java
    // minus 减时间
    LocalDateTime now = LocalDateTime.now();
    DateTimeFormatter dateTimeFormatter = DateTimeFormatter.ofPattern(“yyyy-MM-dd hh-mm-ss”);
    LocalDateTime localDateTime1 = now.minusMinutes(5632);
    System.out.println(“5632分钟前是：” + dateTimeFormatter.format(localDateTime1));

7.Instant类

Instant是时间戳

Instant类似Date，可以和Date相互转换

  1. Instant转Date

    ```java
    // Instant –> Date
    Instant now = Instant.now();
    Date date = Date.from(now);

  2. Date转Instant

    ```java
    // Date –> Instant
    Instant now = Instant.now();
    Instant instant = date.toInstant();
