Speed Down总结

发表于 2022-08-17 更新于 2022-08-16

背景

创建Quad垂直面作为背景，Quad带有3个组件：Mesh Filter、Mesh Renderer、Mesh Collider

创建Unity/Texture类型的Material，选择一个图片修改材质。在Quad的Mesh Renderer组件中添加这个材质

给Quad添加脚本，脚本中获取到Mesh Renderer组件中的材质，修改材质的Offset参数实现移动的效果

Mesh Filter组件

Mesh Filter 组件包含对网格的引用。该组件与同一个游戏对象上的 Mesh Renderer 组件配合使用；Mesh Renderer 组件渲染 Mesh Filter 组件引用的网格。

Mesh Renderer组件

Mesh Renderer 组件用于渲染网格。该组件与同一个游戏对象上的 Mesh Filter 组件配合使用；Mesh Renderer 组件渲染 Mesh Filter 组件引用的网格。

Mesh Collider组件

网格碰撞体：采用网格资源并基于该网格构建其碰撞体。

Polygon Collider 2D组件

Polygon Collider 2D组件是多边形碰撞体

如果希望此碰撞体由附加的2D复合碰撞体(Composite Collider 2D)使用，可以启用Used by Composite,其他属性会从2D盒型碰撞体组件中消失,因为这些属性现在由附加的2D复合碰撞体控制。从2D盒型碰撞体消失的属性为Material, Is Trigger, Used By Effector和Edge Radius.
