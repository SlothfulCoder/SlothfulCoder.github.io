发表于 2022-08-16 更新于 2021-06-19

知识点：什么是jdk、数据类型、类型转换、常见编码、运算符、2进制、位运算、控制结构（分支结构、循环结构）、数组、冒泡排序

1.jdk包含内容

jdk = jre + 开发工具集（如javac，java）

jre = jvm + Java标准类库

2.Java基础注意事项

  1. 一个源文件最多只能有一个public类，其他类的个数不限。
  2. 如果源文件包含一个public类，则文件名必须按该类名命名。
  3. 编译后，每一个类都有一个字节码文件（.class）文件。

4.数据类型

共八种基本数据类型byte,short,int,long,float,double,char,boolean。

6.常见字符编码

  1. ASCII编码表：共128个字符，每个字符用1个字节表示；其实1个字节可以表示256个字符，但ASCII编码表只用了128个。
  2. Unicode：为了引入更多的字符，每个字符都使用2个字节表示。
  3. utf-8：可以理解为对Unicode编码表的优化，字母使用1个字节，汉字使用3个字节。
  4. gbk：字母使用1个字节，汉字2个字节
  5. gbk2312：可以表示汉字，gbk2312 < gbk
  6. big5码：繁体中文，台湾、香港

7.自动类型转换注意事项

  1. byte，short和char之间不会互相自动转换。
  2. byte，short，char他们三者在计算时首先转换为int类型。
  3. 把精度大的数据类型赋值给精度小的数据时，会报错。
  4. 把具体的数赋值给byte时，先判断该数是否在byte范围内，如果是就不会报错。

8.String和基本数据类型互转

  1. 基本数据类型转String类型：基本类型+””即可。
  2. String类型转基本数据类型：使用包装类的parseXX方法即可。

11.&&和&的区别

  1. &&叫短路与，如果第一个条件为false，后面的条件不再判断
  2. &叫逻辑与，如果第一个条件为false，后面条件仍然会判断

另外，||和|类似。

补充，^叫逻辑异或，当左右不同时，结果为true，否则为false。

13.四种进制

  1. 二进制：满2进1，以0b或0B开头
  2. 十进制：满10进1
  3. 八进制：满8进1，以0开头表示
  4. 十六进制：满16进1，A(10)-F(15)，以0x或0X开头表示，此处的A-F不区分大小写。

14.原码、反码、补码（重点 难点）

记住一下几点

  1. 二进制最高位是符号位：0表示正数，1表示负数。
  2. 正数的原码，反码，补码都一样（三码合一）。
  3. 负数的反码=它的原码符号位不变，其他位取反（0->1,1->0）
  4. 负数的补码=它的反码+1，负数的反码=负数的补码-1。
  5. 0的反码，补码都是0。
  6. java没有无符号数，换言之，java中的数都是有符号的。
  7. 在计算机运算的时候，都是以补码的形式来运算的。
  8. 当我们看运算结果的时候，要看他的原码。

15.位运算

java中有7个位运算符（&、|、^、~、>>、<<和>>>）

  1. 按位与&：两位全为1，结果为1，否则为0。
  2. 按位或|：两位有一个为1，结果为1，否则为0
  3. 按位异或^：两位一个为0，一个为1，结果为1，否则为0
  4. 按位取反~：0->1,1->0
  5. 算术右移>>：低位溢出，符号位不变，并用符号位补溢出的高位，本质是/2
  6. 算术左移<<：符号位不变，低位补0，本质是*2
  7. 逻辑右移（无符号右移）>>>：低位溢出，高位补0

特别说明：没有<<<符号。
