发表于 2022-08-16 更新于 2022-07-15

关键字：9个回调方法、GameObject、Transform、Vector3、Quaternion、Time、Mathf

脚本继承关系

脚本 -> MonoBehaviour -> Behaviour -> Component -> Object

9个回调方法

多个脚本中Awake、Start方法的执行顺序可以设置：Edit -> Project Settings -> Script Execution Order

  1. Awake：游戏开始执行一次
  2. Start：在OnEnable执行后执行，只会执行一次
  3. OnEnable：组件每次被设为可用时执行
  4. FixedUpdate：每隔一段时间执行一次，默认时0.02s
  5. Update：每帧执行一次
  6. LateUpdate：每帧执行一次，在update后执行
  7. OnGUI：渲染旧版UI，每帧大约执行两次
  8. OnDisable：每当组件设为不可用时，执行一次
  9. OnDestroy：当脚本组件被销毁，或整个游戏对象销毁时调用一次

GameObject

  1. activeSelf：表示当前游戏对象的激活状态
  1. SetActive(bool)：设置当前游戏对象的激活状态
  1. name：名字
  1. tag：标签
  1. layer：层
  1. GetComponent：获取当前游戏对象组件
  1. AddComPonent：添加一个组件到游戏对象上，并返回这个组件
  1. static Find：通过name找游戏对象，不能找到非激活的对象，重名可能找不到，对象多消耗性能
  1. static FindWithTag：通过tag找游戏对象，不能找到非激活的对象，重名可能找不到
  1. static FindGameObjectsWithTag：通过tag找游戏对象对象数组，不能找到非激活的对象

Transform

  1. position：世界坐标(以世界的000为原点)

  2. localPosition：本地坐标（相对于父物体的坐标）

  3. eulerAngles：世界欧拉角

  4. localEulerAngles：本地欧拉角

  5. rotation：四元数

  6. localRotation：本地四元数

  7. localScale：缩放

  8. forward：自身前方向量

  9. up：自身上方向量

  10. right：自身右方向量

  11. parent：父对象，还是Transform类型，如果没有父对象，得到null

  12. root：根对象，还是Transform类型，如果没有根对象，得到自己

  13. childCount：子对象个数

  14. GetChild：获取子物体

  1. SetParent：设置父物体
  1. Find：从子对象中找某一个对象，可以找到非激活的对象
  1. Translate：移动
  1. Rotate：自转
  1. RotateAround：绕某个点沿某个轴旋转
  1. LookAt：看向某个对象

Vector3

  1. Vector3是一个三维向量，但也不一定用于表示三维向量，只要是三个浮点数组成的数据，都可以用Vector3去表示
  2. normalized：当前向量的单位向量
  3. Normalize()：同上
  4. magnitude：向量的模
  5. sqrMagnitude：模的平方
  6. static forword：世界的前方，(0, 0, 1)
  7. static right：世界的右方，(1, 0, 0)
  8. static up：世界的上方，(0, 1, 0)
  9. static back：世界的后方，(0, 0, -1)
  10. static left：世界的左方，(-1, 0, 0)
  11. static down：世界的下方，(0, -1, 0)
  12. static zero：(0, 0, 0)
  13. static one：(1, 1, 1)
  14. Distance(Vector3, Vector3)：求两点距离
  15. static float Angle(Vector3, Vector3)：求两个向量夹角
  16. static float Dot(Vector3, Vector3)：求两个向量点乘|a||b|cos
  17. static Vector3 Cross(Vector3, Vector3)：求两个向量叉乘（法向量）
  18. static Vector3 Lerp(Vector3, Vector3, float)：插值函数

Quaternion

  1. static identity：相当于欧拉角中的000
  1. static LookRotation：将一个向量转换为这个方向所代表的四元数
  1. static Lerp：插值函数

Time

  1. time：从游戏开始到此时，所花费的时间
  2. deltaTime：从上一帧结束到当前帧开始，所花费的时间
  3. fixedDeltaTime：固定间隔时间，默认0.02
  4. timeScale：时间缩放，默认1

Mathf

Abs：求绝对值

Lerp(float, float, float)：插值函数

Clamp：限制值

Pow：求某个数字的某次方
