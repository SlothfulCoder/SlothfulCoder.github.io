发表于 2022-08-16 更新于 2021-09-12

关键字：JSON

JSON

JSON(JavaScript Object Notation, JS 对象简谱) 是一种轻量级的数据交换格式。它基于 ECMAScript (欧洲计算机协会制定的js规范)的一个子集，采用完全独立于编程语言的文本格式来存储和表示数据。

json现在多用于存储和交换文本信息的语法，进行数据的传输，JSON 比 XML 更小、更快，更易解析

json数据格式

jackson

导入jar包jackson-annotations-2.2.3.jar，jackson-core-2.2.3.jar，jackson-databind-2.2.3.jar

jackson是一种JSON解析器，可以把JavaBean、集合等数据格式和json相互转换，常见的josn解析器还有Jsonlib，Gson，fastjson

用法

  1. 创建ObjectMapper对象
  2. 用ObjectMapper对象的方法实现转换

ObjectMapper的方法

  1. String writeValueAsString(Object var1)：把JavaBean、集合等对象转换为json字符串
  2. void writeValue(File var1, Object var2)：将obj对象转换为JSON字符串，并保存到指定的文件中
  3. void writeValue(Writer var1, Object var2)：将obj对象转换为JSON字符串，并将json数据填充到字符输出流中
  4. void writeValue(OutputStream var1, Object var2)：将obj对象转换为JSON字符串，并将json数据填充到字节输出流中
  5. T readValue(String var1, Class<T> var2)：把json格式数据转换为对应的对象

注解控制

有些JavaBean的属性我们不想格式化，或者需要对形式做出修改

  1. @JsonIgnore：排除属性

  2. @JsonFormat：属性值得格式化

相应json格式数据

服务器响应的数据，在客户端使用时，要想当做json数据格式使用。有两种解决方案：

  1. $.get(type):将最后一个参数type指定为”json”

  2. 在服务器端设置MIME类型
